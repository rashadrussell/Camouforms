Camouforms is a drag & drop form builder.

== Description ==

Generate custom forms for your Wordpress themes and track submissions. This is the first version release of Camouforms, and more features will be added. Leave comments on what improvements and additional features that you would like to see in future releases.


== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/camouforms` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress


== Screenshots ==

1. Dashboard to view user submissions and your forms.
2. Drag & drop form builder.
